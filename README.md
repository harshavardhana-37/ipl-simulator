# 🏏 IPL Simulator

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://your-vercel-link.vercel.app)

A dynamic IPL match simulator built using React that allows users to simulate match outcomes and view a real-time updated points table.

## 🚀 Live Demo

👉[ https://your-vercel-link.vercel.app](https://ipl-simulator-seven.vercel.app/)

## 📌 Features

- 🎮 Simulate IPL matches between teams
- 📊 Real-time points table updates
- 🔄 One-click random match simulation
- ⚡ Instant calculation of standings
- 🎨 Clean and minimal dark UI

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Language:** JavaScript
- **Styling:** Inline CSS
- **Deployment:** Vercel

---

## 🧠 How It Works

- Each match allows you to select a winner
- Points system:
  - Win → +2 points
  - Loss → 0 points
- Table updates dynamically after every selection
- You can randomize all matches instantly for quick simulation
